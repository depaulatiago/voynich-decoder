## ğŸ”¬ Methodology

The Voynich Decoder uses a multi-method approach:

1. **Statistical Analysis** - Frequency analysis, n-grams, entropy, Zipf's law
2. **Embeddings** - Word2Vec and sentence transformers for semantic patterns
3. **Language Comparison** - Jensen-Shannon Divergence with historical corpora
4. **Temporal Evolution** - Token usage patterns across manuscript folios
5. **Visual Overlay** - Manuscript image analysis with token annotations
6. **LLM Hypothesis Generation** - AI-assisted pattern interpretation
